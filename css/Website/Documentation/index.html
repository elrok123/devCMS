A) HTML Structure - top

  • The .wrapper class sets the width of the content within it to 970 and vertically aligns it so you get the effect that the page is centered. creating a div that is 100% of the page and then putting wrapper INSIDE it is the preferred practice for new page elements.
  • The .shapes class applies a semi-transparent background which gives the background behind it an effect. For example see the home page above the plan boxes.
  • The #table ID creates a CSS table. The .row class creates a row (add .summary for the summary row). The .col class creates a cell (add .heading to add a gradient background if you want the cell to serve as a label for the rest of its row. For an example of this see the plans page.
  • Use .col by itself to create boxes (these will take us 1/3 of the page's width). Apply .first to the first and .last to the last as well. This is shown on the home, plans, and vps pages.
  • To create a box similar to the styling of the 3 plan boxes on the home page, create a .box div. Inside of it, put a .servers div (leave empty) and a .content div. Inside of the .content div you may add header tags, p tags, and ul tags which will be styled appropriately.

B) CSS Files and Structure - top

The theme has 4 CSS files.

  • main.css - Simply includes the others.
  • normalize.css - Fixes cross browser discrepancies.
  • default.css - Most of the actual stylings.
  • fonts.css - Application of the web font.

Below are some useful classes for common elements.

Text Colour Classes

  • .blue - <span class="blue>text</span> This will make the text blue
  • .orange - <span class="orange>text</span> This will make the text orange
  • .green - <span class="green>text</span> This will make the text green

Button Classes

  • .smallButton - <div class="smallButton>Button text</div> Creates a small green button
  • .smallButton.orange - <div class="smallButton orange">Button text</div> Creates a small orange button
  • .smallButton.blue - <div class="smallButton blue">Button</div> Creates a small blue button
  • .fatButton - <div class="fatButton">Button</div> Creates a large fat button
